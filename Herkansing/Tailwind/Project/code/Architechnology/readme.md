# Architechnology - Website Basis

Een moderne en futuristische website voor het architectenbureau **Architechnology** dat zich specialiseert in het bouwen van futuristische woningconcepten.

## 📁 Mappenstructuur

```
Architechnology/
├── index.html              # Startpagina (Home)
├── info.html              # Informatie over het bedrijf
├── producten.html         # Woningconcepten & producten
├── realisaties.html       # Voltooide projecten
├── contact.html           # Contactformulier
├── sollicitatie.html      # Werkgelegenheid & sollicitaties
├── css/
│   └── styles.css        # Custom CSS styling
├── js/
│   └── script.js         # JavaScript functionaliteit
└── img/                  # Map voor afbeeldingen
```

## 📝 Customisatie

### Kleuren Aanpassen
Edit `css/styles.css` en pas de Tailwind klassen aan:
- `from-purple-600` → andere paarse tint
- `to-cyan-600` → andere cyan tint

### Content Aanpassen
Alle teksten kunnen direct in de HTML bestanden worden aangepast

### Afbeeldingen Toevoegen
1. Plaats afbeeldingen in de `img/` map
2. Update de `src` attributen in HTML
