Collapse

Voorbeeld op getbootstrap.com/docs/5.3/components/collapse/

Om dit voorbeeld te laten werken moet je enkel het script van de Bootstrap bundel toevoegen.

Dit doe je best onderaan je pagina, voor je /body zodat de rest van je pagina zeker geladen is.

Je doet dit best op een <button> element waarop je 2 data- attributen toepast :

data-bs-target
verwijst naar het element dat je wil aanpassen
data-bs-toggle="collapse"
omdat je een collapse-actie wil uitvoeren als hier wordt op geklikt.

Dit wordt dan toegepast op een element met class .collapse en met de id waarnaar je verwees in data-bs-target.
